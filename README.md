=======
LocalConnect

LocalConnect is a simple web project that helps you find local service providers like mechanics, plumbers, electricians, and carpenters. It's built to connect customers who need a job done with the skilled workers who can do it.

This was built as a miniproject.

What It Can Do (Features)

This website has three main types of users: Customers, Service Providers, and an Admin. Each user gets their own dashboard to manage their tasks.

For Customers

Find Help: Search for different types of service providers in your area.

Book Appointments: Easily book a time with the provider you choose.

Leave Reviews: Share your experience by writing a review for the service provider.

Raise a Complaint: If there's a problem, you can raise a complaint for the admin to see.

For Service Providers

Create a Profile: Sign up and fill out your profile with your skills and details.

Get Verified: Your profile must be approved by the admin before customers can see it.

Manage Bookings: See all your appointment requests from customers.

Confirm or Reject: You can choose to accept or reject new appointments based on your schedule.

Handle Reviews: See all the reviews customers have left for you and write replies to them.

For the Admin

Approve Providers: Review new service providers who sign up and verify their details.

Manage Complaints: See and handle all complaints filed by customers.

View All Providers: Search for any service provider to see their full information and status.

Oversee the Site: Keep an eye on the platform to make sure it's running smoothly.

Technologies Used

Frontend: HTML, CSS, JavaScript

Backend: PHP

Database: MySQL

Server: XAMPP (Apache + MySQL)

How to Run This Project

To get this project running on your own computer, just follow these steps:

Download the Project: Clone or download the ZIP of this project to your computer.

Add to XAMPP: Unzip the file and put the entire project folder inside your XAMPP htdocs directory.

(Usually found at C:\xampp\htdocs\)

Start XAMPP: Open your XAMPP Control Panel and start both the Apache and MySQL services.

Set up the Database:

Open your web browser and go to http://localhost/phpmyadmin.

Create a new, empty database. (You can name it something like localconnect_db).

Click on your new database in the list, then click the Import tab.

Choose the .sql file that is included in this project folder and click "Go". This will set up all the tables you need.

All Done!
